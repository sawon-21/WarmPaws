# WarmPaws

WarmPaws is a cozy winter pet care platform for browsing seasonal services, booking care, and managing a simple user profile.

## Live URL

## Features

- Winter themed home page with Swiper hero slider
- Service cards loaded from local JSON data
- Protected service details and profile routes
- Email/password login, registration, Google login, and logout
- Password validation and password visibility toggle
- Functional forgot password page with Gmail redirect
- Profile update form using Firebase `updateProfile()`
- Toast notifications and subtle AOS animations
- Responsive layout for mobile, tablet, and desktop

## Packages Used

- React
- React Router
- Firebase
- Swiper
- AOS
- react-hot-toast
- lucide-react
